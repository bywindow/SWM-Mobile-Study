# 🧨 UI?
---
**UI**는 앱의 사용자가 상호작용 할 수 있는 모든 것을 의미한다. Android에서는 **UI Components**(레이아웃 객체, UI Control)를 미리 빌드된 형태로 제공하여 앱의 그래픽 UI를 빌드하는 데에 사용한다. 또한 대화상자, 알림, 메뉴를 위한 다른 UI모듈도 제공한다.
